get /coaches/{id}
System shows coach id's plcoachayer page. (coachesIdGet)
User requests to get coach info.

Path parameters

id (required)
Path Parameter — Numeric ID of the coach id to get

Return type

CoachFull

Example data

Content-Type: application/json
{
  "team name" : "Maccabi Haifa",
  "nationality" : "Isreali",
  "img-url" : "https://cloudinary.com/john_kary",
  "weight" : 78,
  "common name" : "Messi",
  "birth country" : "France",
  "full name" : "Barak Bachar",
  "birth date" : "2000-01-23",
  "height" : 182
}
